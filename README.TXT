
PROJECT TITLE: Assegnamento 1
PURPOSE OF PROJECT: Simulazione in linguaggio java di un sistema automatizzato di rilevamento di entrata e uscita auto con la visualizzazione di dati sul traffico e sui posti disponibili.

VERSION or DATE: 16 Maggio 2014
HOW TO START THIS PROJECT: Da terminale, eseguire con il comando “java Simulator” dalla cartella "bin" all'interno del progetto.

AUTHORS: Vincenzo Arceri - Matteo Calabria - Pietro Musoni - Carlo Tacchella

USER INSTRUCTIONS: Nella finestra "Detector" (a sinistra dello schermo), cliccare sul bottone "Enter" per simulare l'entrata di una macchina e cliccare sul bottone "Exit" per la simulazione dell'uscita di un'auto dal parcheggio. Il bottone "Add an Hour" permette di aggiungere un'ora all'orologio di sistema (per il calcolo della media). 
La finestra "processing unit" mostrerà il totale di macchine entrate nel parcheggio e le ore passate da quando è partito il sistema, di default parte con 1 (inizia dalla prima ora della giornata). 
La finestra "Monitor" mostrerà la media di macchine entrate ogni ora e il numero di posti disponibili nel parcheggio. Durante tutto il processo veranno stampati sul terminale tutte le operazioni eseguite dalle tre unità.
